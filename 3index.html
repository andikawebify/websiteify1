Apa Itu Website?

Penjelasan Singkat

Sebuah website adalah kumpulan halaman digital yang saling terhubung dan dapat diakses melalui internet menggunakan sebuah browser (seperti Chrome, Firefox, atau Safari). Website dapat berisi teks, gambar, video, suara, maupun aplikasi interaktif lainnya.

Website biasanya memiliki alamat khusus yang disebut URL atau domain name, contohnya www.google.com. Website bisa dibuat untuk berbagai tujuan seperti edukasi, hiburan, jual beli online, portofolio pribadi, hingga bisnis perusahaan.

Jenis-Jenis Website

Beberapa jenis website yang umum dijumpai antara lain:

  • Website Pribadi (Blog, Portofolio)
  • Website Bisnis dan Perusahaan
  • Website E-Commerce (Toko Online)
  • Website Pendidikan
  • Website Media & Berita
